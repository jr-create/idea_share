## 视觉样式验证

### 视觉层次与重点突出
- [ ] 标题层级明显：验证页面标题、section标题和内容标题有明显的字号、权重和颜色区分
- [ ] 重点元素突出：验证重要按钮（如提交创意、删除按钮）有足够的视觉权重
- [ ] 内容区分度高：验证不同类型的内容（如项目描述、需求列表、创意列表）之间有明显的视觉分隔
- [ ] 状态指示清晰：验证项目状态、需求状态等状态信息的视觉表示直观

### 交互反馈与动效
- [ ] 悬停效果丰富：验证按钮、链接等可交互元素的悬停效果有层次感
- [ ] 点击反馈明显：验证按钮点击时的视觉反馈明显
- [ ] 加载状态有指示：验证页面加载、表单提交等操作时有加载动画
- [ ] 过渡动画平滑：验证模态框的显示/隐藏、内容的展开/收起等操作有平滑的过渡动画
- [ ] 滚动效果生动：验证页面滚动时有视差效果或渐入效果

### 色彩与品牌一致性
- [ ] 色彩方案丰富：验证整体色彩方案不单调，有辅助色和强调色
- [ ] 品牌识别度高：验证形成了统一的品牌色彩体系
- [ ] 色彩对比足够：验证文本与背景的对比度足够，影响可读性
- [ ] 色彩使用一致：验证不同功能模块的色彩使用一致

### 响应式设计与适配
- [ ] 移动端布局合理：验证在小屏幕设备上，内容布局不拥挤，元素间距充足
- [ ] 导航栏适配良好：验证移动端导航栏适配为汉堡菜单
- [ ] 图片适配得当：验证图片在不同屏幕尺寸下的显示效果一致
- [ ] 表单元素适配合理：验证表单输入框、按钮等元素在移动端的尺寸合适
- [ ] 文字大小适配：验证不同屏幕尺寸下的文字大小合理调整

### 布局与空间利用
- [ ] 留白充足：验证页面元素之间的留白足够，不视觉拥挤
- [ ] 对齐统一：验证元素对齐方式一致，整体整洁
- [ ] 网格系统规范：验证布局有统一的网格系统，元素排列规则
- [ ] 空间分配合理：验证重要内容与次要内容的空间分配比例得当

### 图标与视觉元素
- [ ] 图标风格统一：验证使用的图标风格一致，整体视觉协调
- [ ] 图标尺寸规范：验证不同位置的图标尺寸一致，标准化
- [ ] 视觉元素丰富：验证有适当的装饰性元素或图形元素，页面不过于简洁
- [ ] 图标与文字搭配协调：验证图标与文字的比例和位置关系协调

### 表单与输入体验
- [ ] 表单样式美观：验证表单输入框、下拉菜单等元素的样式有设计感
- [ ] 输入反馈明显：验证表单验证的视觉反馈明显
- [ ] 错误提示样式突出：验证错误信息的样式突出，不被用户忽略
- [ ] 表单布局清晰：验证表单元素的排列和分组清晰，影响用户填写体验

### 卡片与容器设计
- [ ] 卡片样式丰富：验证项目卡片、创意卡片等容器的样式有层次感
- [ ] 阴影效果自然：验证卡片阴影效果自然，营造出悬浮感
- [ ] 边框与圆角一致：验证不同容器的边框样式和圆角半径一致
- [ ] 卡片内容布局合理：验证卡片内部的内容布局合理，信息展示不混乱

- [ ] 字体选择多样：验证整体字体选择不单一，有字体层次
- [ ] 行高与字间距合理：验证部分文本的行高和字间距设置得当，影响可读性

- [ ] 离线状态处理：验证有针对网络离线状态的视觉提示
